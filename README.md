# Guilherme Antoniacomi | Analytics Engineering Portfolio

This repository contains the source code for my professional portfolio and curriculum vitae, hosted via GitHub Pages. 

🌐 **Live site:** [GuiAntoniacomi.github.io](https://GuiAntoniacomi.github.io)

## Under the Hood

I built this site to reflect my daily data engineering workflow: clean, functional, and architecture-first. There are no heavy frameworks or bloated site builders—just straight code.

* **Tech Stack:** HTML5, Tailwind CSS (via CDN), and Vanilla JavaScript.
* **Key Features:**
  * **Bilingual Architecture:** Native PT-BR and EN support handled entirely via Vanilla JS and CSS attribute selectors, avoiding page reloads.
  * **Dynamic Assets:** Resume download links update automatically based on the user's active language state.
  * **Custom UI:** Terminal/IDE aesthetic featuring CSS-generated film grain, custom scroll tracking, and pure CSS layout grids.
